# Instructions to generate release keystore:
# 1. Open terminal/command prompt
# 2. Navigate to android/app directory
# 3. Run the following command:
#    keytool -genkeypair -v -storetype PKCS12 -keystore my-upload-key.keystore -alias my-key-alias -keyalg RSA -keysize 2048 -validity 10000
# 4. Enter the store password when prompted
# 5. Enter the key password when prompted
# 6. Fill in the certificate information
# 7. Update local.properties with the correct passwords

# Important security notes:
# - Never commit keystore files to version control
# - Store passwords securely
# - Backup your keystore file safely
# - Use different passwords for store and key
